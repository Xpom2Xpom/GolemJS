overview golem API

Utils:

golem.utils.StringBuilder

remove(str: string, sym: string) => string

Remove all symbols sym from string str.
golem.utils.StringBuilder.remove('abcb', 'b') => 'ac'

removeArr(arr: array[string], sym: string) => array[string]

Remove all symbols sym from all string of array arr.
golem.utils.StringBuilder.removeArr(['abc1', '11a'], '1') => ['abc', 'a']

parse(str: string, sep: string, pos: number) => string

Get some parth of string str splited by separator sep on position pos
golem.utils.StringBuilder.parse('aaa.bbb.ccc', '.', 1) => 'aaa'

parse(str: string, sep: string, pos: array[number]) => array[string]

Get some array of strings for string str splited by separator sep on positions pos
golem.utils.StringBuilder.parse('aaa.bbb.ccc', '.', [2, 3]) => ['bbb', 'ccc']

parseArr(arr: array[string], sep: string, pos: number) => array[string]

The same as parse but for array of strings
golem.utils.StringBuilder.parseArr(['aaa.bbb', 'ccc.ddd'], '.', 2) => ['bbb', 'ddd']

build(template: string, value: string) => string

Replase sym (default {1}) in template by value
golem.utils.StringBuilder.build('aaa {1} bbb', 'sss') => 'aaa sss bbb'

build(template: string, value: string, value2: string) => string

Replase sym {1} and sym2 {2} - and anothers {1} - {n} - in template by values
golem.utils.StringBuilder.build('aaa {1} {2} bbb', 'sss', 'ddd') => 'aaa sss ddd bbb'
golem.utils.StringBuilder.build('aaa {1} {2} {3} bbb', '111', '222', '333') => 'aaa 111 222 333 bbb'

buildArr(template: string, value: string, value2: string, ...) => string

The same as build but for array of templates
golem.utils.StringBuilder.buildArr(['aaa {1} bbb', 'ccc {1} ddd'], 'sss', 'fff') => ['aaa sss bbb', 'ccc fff ddd']
golem.utils.StringBuilder.buildArr(['aaa {1} {2} bbb', 'ccc {1} {2} ddd'], ['111', '222'], ['333', '444']) => ['aaa 111 222 bbb', 'ccc 333 444 ddd']

buildArrStepByStep(template: string, value: array[string], value2: array[string], ...) => array[string]

Return array of strings building from templates using for each element from each array - for {1} first element from all arrays, for {2} - second ...
golem.utils.StringBuilder.buildArrStepByStep('aaa {1} {2} {3}', [1,2,3], [4,5,6], [7,8,9]) => ["aaa 1 4 7", "aaa 2 5 8", "aaa 3 6 9"]
